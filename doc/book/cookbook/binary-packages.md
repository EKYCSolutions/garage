Garage is also available in binary packages on:

## Alpine Linux

```bash
apk install garage
```

## Arch Linux

Garage is available in the [AUR](https://aur.archlinux.org/packages/garage).

## FreeBSD

```bash
pkg install garage
```

## NixOS

```bash
nix-shell -p garage
```
